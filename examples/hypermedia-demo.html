How it works

The src attribute on a div (or any non-native element) triggers a hypermedia load. Any query parameters in the URL are automatically parsed and injected as implicit macro arguments.

  • $.name: Accesses the parameter from the URL context. If a local macro defines a name argument, it will shadow this value.
  • $query.name: Explicitly accesses the URL parameter, bypassing any local macro shadowing.
  • Scope Bubbling: The engine searches up the component tree to find the nearest match, ensuring that nested components can still reach global query parameters.
